# LaunchProof

Local-first MVP for turning AI-built prototypes into production-readiness reports.

## Run locally

This is a Next.js app, so it cannot be opened by double-clicking an HTML file.
Start the development server first:

```bash
npm install
npm run dev
```

Then open:

```txt
http://localhost:3000
```

## If `npm` is not found

Install Node.js and npm in your local terminal, then rerun the commands above.
On Ubuntu/WSL:

```bash
sudo apt update
sudo apt install -y nodejs npm
```

Project data is stored locally in `.data/launchproof.json`.
